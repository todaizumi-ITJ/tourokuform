## 技術メモ

- 照合スクリプト: Python Flask + BeautifulSoup4 + openpyxl
- 既存DB: db_human.json（32,742件）、cp_numsフィールド付き
- IPN→CP解決: db_ipn.json（16,853件）
- RH補完: db_mandate_rh.json（58権利者 × cp_numsリスト）
- 包括和解: db_settled.json（cp_nums: [1,2,3,4,5,6,8]）
- 氏名正規化: Unicode NFKC + 異体字統一（20字種）
- 住所正規化: 全角→半角、丁目/番地/号→ハイフン、前20/12文字比較
- Excel出力: 4シート（全件 / 通知する / 通知しない / 通知しない詳細）
- 全列取得: registration_type〜pdf_filename（列B〜T）すべて取得・出力
